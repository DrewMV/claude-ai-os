# Tag Taxonomy

Canonical tag vocabulary for this vault. Use these tags — do not invent new ones without adding them here first.

## Workspace Tags

```
work
research
personal
```

## Status Tags

```
active          # currently in progress
paused          # on hold
completed       # done
archived        # no longer relevant
```

## Content Type Tags

```
concept         # idea, pattern, or principle
decision        # a recorded choice
question        # open question
reference       # external source summary
person          # individual
tool            # software, service, or system
process         # a repeatable workflow or method
```

## Adding New Tags

1. Add the tag to this file under the appropriate section
2. Include a one-line description if the meaning is not obvious
3. Run `/tag-taxonomy` to normalize existing pages to use the new canonical tag
